PHP 5.3 Closures

Submitted by Rayhan Muktader on Wed, 10/10/2012 - 12:34

PHP closure allow a function to enclose a variable and retain a reference to that variable even after the variable falls out of scope. 

<?php

function getClosure()
{
    $localVar = 1;
    
    return function () use ( &$localVar )
    {
        return ++$localVar;
    };
    
}


$myClosure = getClosure();
echo $myClosure() . PHP_EOL; // prints: 2


$notMyClosure = getClosure();
echo $notMyClosure() . PHP_EOL; // prints: 2


echo $myClosure() . PHP_EOL; // prints: 3
